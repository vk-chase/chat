# chat
QBCore chat design



## Major changes

### Moved and saved chat position

* Added a **Move chat** button.
* The chat can now be repositioned on screen.
* Its position saves, so it stays where you put it after restarting.

### Fixed chat history overlapping the input

* Cleaned up the layout so the chat history no longer runs down over the input bar.
* The overall chat area behaves a lot cleaner now.

### Better scaling support

* Adjusted the layout so it works better across:

  * 1080p
  * 1440p
  * 4K

### Added chat color customization

* Added **Box color** and **Text color** controls beside the move button.
* Reworked the first version into a more game-style color picker instead of plain RGB sliders.

### Added live style controls

* Color changes now update live while adjusting.
* Added controls for:

  * color
  * transparency
  * glow strength
